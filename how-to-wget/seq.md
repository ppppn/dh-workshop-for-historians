## 数字を作る

さて、画像によって変わる部分を変数に置き換えることができました。次は `$i` に代入される数字を増やしたいですよね。

Linuxには標準で seq というコマンドがあります。軽く実行してみましょう:

```
$ seq 3 10
3
4
5
6
7
8
9
10
```

意味はもうわかりますよね?次は seq コマンドのもっとも簡単な書式です。

```
seq 開始番号 終了番号
```

今回、 `i` に入る数字は4桁で、数字のない桁は0で埋めていました。余った桁を0で埋めることをゼロパディングといいます。seqは`-w`オプションをつけると、ゼロパディングを解釈してくれます:

```
$ seq -w 0003 0010
0003
0004
0005
0006
0007
0008
0009
0010
```

ゼロパディングが必要なときもそうでない時もあるので、そのあたりはうまくやってください。
